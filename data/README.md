# Data folder

The app starts with a clean `library.json` so each device can build its own private music collection.

If the app grows, this folder is a natural place to split data into files such as:

- `setlists.json`
- `indexes.json`
- `cards.json`

Keep private or copyrighted PDFs local to your device or in the `music/` folders, then add them through the app or point to them from `library.json` if you choose to edit by hand.
